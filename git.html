USEFUL GIT / GITHUB Commands

git init project for version control (do this from within the folder to be initialised)
git status see what changes have been made
git add {filename} add file into staging
git commit -m "message" commit change to git and add a relevant comment to identy changes made
git log log of all commits
git remote add origin {url} sets up connection to git-hub account
git push origin {branchname} sends code to branch on git-hub
git remote -v provides detail of target url
git clone {url) takes a copy from github and loads it on your pc .... NB puts the code in a folder
rm -rf .git Trouble shooting - delete local git repository
git remote -v Shows you the url of github if you are connected
